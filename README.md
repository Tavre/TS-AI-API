# TS-AI API Documentation

TS-AI 开发者 API 文档源码仓库（基于 VitePress）。

- **文档源码**: `docs/` 目录
- **在线预览**: `npm run dev`
- **构建部署**: `npm run build`
- **部署指南**: [Vercel 部署](DEPLOY.md) | [宝塔部署 (BT)](DEPLOY_BT.md)

## 快速开始

1. **安装依赖**
   ```bash
   npm install
   ```

2. **本地启动**
   ```bash
   npm run dev
   # 访问 http://localhost:5173
   ```
