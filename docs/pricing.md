# 费用说明

## 扣费规则

| 项目         |  消耗   | 说明                             |
| ------------ | ------: | -------------------------------- |
| 每次图像生成 | 10 积分 | 约 0.10 元                       |
| 失败自动退款 |   是    | 如上游引擎报错，积分自动退回     |

## 积分扣除优先级

```
临时积分（按过期时间升序） --> 主账户积分
```

系统优先扣除**即将过期的临时积分**，最后扣除主账户永久积分。

## 获取积分

| 途径       | 类型     | 说明                      |
| ---------- | -------- | ------------------------- |
| 每日签到   | 临时积分 | 赠送临时积分，有过期时间  |
| 充值       | 主账户   | 永久积分                  |
| 兑换码     | 主账户   | 永久积分                  |
